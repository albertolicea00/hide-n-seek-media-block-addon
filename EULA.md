## 3. Description of the Extension

Hide & Seek is a cross-browser extension for Google Chrome, Mozilla Firefox, and compatible Chromium-based browsers. It automatically blurs, blacks out, or replaces with placeholder images any media elements (images, videos, iframes) on websites the User visits, based on user-defined configuration. All processing occurs **entirely locally** within the User's browser.

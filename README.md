# Hide N Seek - Media Blocking Extension

**Hide N Seek** is a high-performance browser extension designed to help you browse the web safely and discreetly. It automatically hides sensitive images, videos, and thumbnails, allowing you to navigate even the most explicit websites without seeing anything you don't want to.

## 🛡️ Features

- **Instant Protection**: Automatically blurs images, videos, iframes, and background images as they load.
- **Deep Obfuscation**: Uses extreme blurring and grayscale filters to ensure content is completely unrecognizable, preserving your privacy and peace of mind.
- **Click-to-Reveal**: See something you're interested in? Just click once to reveal the specific piece of media instantly.
- **Layout Preserving**: Unlike other blockers that break websites, Hide N Seek maintains the original layout, so the web looks exactly as intended—just blurred.
- **High Performance**: Built with pure CSS and efficient JavaScript for zero lag and minimal CPU usage.
- **Easy Toggle**: Turn the protection on or off with a single click from the extension popup.

## 🚀 How to Install (Developer Mode)

1.  Download or clone this repository.
2.  Open your browser (Chrome, Edge, or any Chromium-based browser).
3.  Go to the extensions page: `chrome://extensions/`.
4.  Enable **"Developer mode"** in the top right corner.
5.  Click **"Load unpacked"** and select the folder containing these files.
6.  Pin the **Hide N Seek** icon to your toolbar for easy access!

## 🛠️ Usage

- **Activate**: Open the popup and toggle the switch to "ON".
- **Reveal Media**: Click on any blurred image or video to see the original content.
- **Deactivate**: Toggle the switch to "OFF" to return to normal browsing.
